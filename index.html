🔧 Setup Koneksi Google Sheets

Ikuti langkah-langkah berikut sekali saja. Setelah selesai, semua laporan akan tersimpan otomatis ke Google Sheets Anda.

Link Aplikasi & QR Code

Bagikan link atau QR code ini ke anggota tim agar mereka bisa membuka dan menginstall aplikasi di HP masing-masing.

Langkah-langkah Setup
  1. 1

    Buat Google Sheet Baru

    Buka sheets.google.com, klik tombol + untuk membuat spreadsheet baru. Beri nama misalnya Damage Report DB.

  2. 2

    Buka Apps Script

    Di Google Sheets, klik menu Extensions → Apps Script. Tab baru akan terbuka.

  3. 3

    Paste Kode Apps Script

    Hapus semua kode yang ada (Ctrl+A lalu Delete), kemudian paste kode di bawah ini. Klik tombol Copy Kode lalu paste ke Apps Script.

  4. 4

    Simpan & Deploy

    Klik ikon 💾 Save, lalu klik Deploy → New Deployment. Pilih type: Web App.

  5. 5

    Atur Permissions

    Set Execute as: Me dan Who has access: Anyone. Klik Deploy, lalu Authorize access jika diminta.

  6. 6

    Copy URL & Paste di Bawah

    Setelah deploy, akan muncul URL yang diawali https://script.google.com/macros/s/.... Copy URL tersebut dan paste di kolom di bawah ini, lalu klik Simpan.

      Keamanan Data

      URL Apps Script hanya disimpan di browser Anda (localStorage). Data laporan tersimpan langsung di Google Sheets milik Anda dan tidak melewati server lain. File yang diupload di tab Berkas disimpan ke folder Google Drive milik Anda (folder PSR BS2 - Berkas dibuat otomatis) dan dibagikan sebagai "Anyone with the link – Viewer" agar semua anggota tim bisa membukanya. Saat pertama kali Deploy, Google akan meminta izin akses Drive — klik Authorize access untuk mengizinkan.
